# IARP

## Installation

Pour installer les dépendances du projet, utilisez la commande suivante à la racine du projet :

```bash
npm install:all
```

Pour upgrader les dépendances du projet, utilisez la commande suivante à la racine du projet :

```bash
npm update:all
```

## Backend

Pour lancer le backend, utilisez la commande suivante à la racine du projet :

```bash
npm run dev
```

## Corriger le code

Pour corriger automatiquement les erreurs corrigeables, tapez la commande suivante :

```bash
npm run lint:fix
```
